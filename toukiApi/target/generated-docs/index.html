Touki API : Documentation

Version: 1.0.0
Auteur: Souleymane Mbaye
Date: 28/11/2020

Api qui permet d’avoir des informations précises sur les vols entre différents pays. Il permet à l’utilisateur de savoir entre autres :

  • La liste de tous les pays

  • La liste des vols entres deux pays donnés

  • Les détails d’un vol bien connu: (l’heure du vol, le prix du billet, etc.)

  • Les cinq vols les moins chers en terme de prix du billet

  • etc.

1. Authentification

Pour accéder à une resource de l’api, il faut d’abord s’authentifier. Après authentification avec des accés valides, un token vous sera généré. Ce token sera fourni pour toutes les autres resources que allez utiliser.
